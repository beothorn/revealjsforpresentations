Reveal JS is a framework for doing slides presentations with plain html and JS

It is an alternative to presentation softwares

It:

  • Runs on any browser
  • Has touch support
  • Support Markdown
  • Only requires a text editor (it's html)
  • Allow Version control
  • Licensed under MIT
  • Lots of features (see official repo)
